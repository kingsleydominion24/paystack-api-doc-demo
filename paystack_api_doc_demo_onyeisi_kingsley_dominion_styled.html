Paystack Payment Gateway Integration Guide

Demo Project by Onyeisi-Kingsley Dominion

1. Overview

This guide walks developers through integrating the Paystack REST API into their web applications for secure online payments. It covers setup, authentication, payment requests, and handling responses.

2. Prerequisites

  • A Paystack account (sandbox or live)
  • API keys from the Paystack Dashboard
  • Basic knowledge of JavaScript or Python
  • A development environment with internet access

3. Authentication

Paystack uses Bearer Token authentication.

Authorization: Bearer YOUR_SECRET_KEY

4. Making a Payment Request

JavaScript (Node.js)

const axios = require('axios');

axios.post('https://api.paystack.co/transaction/initialize', {
  email: 'customer@example.com',
  amount: 500000  // Amount in kobo
}, {
  headers: {
    Authorization: 'Bearer YOUR_SECRET_KEY'
  }
})
.then(res => {
  console.log(res.data);
})
.catch(err => {
  console.error(err);
});

Python

import requests

headers = {
    "Authorization": "Bearer YOUR_SECRET_KEY",
    "Content-Type": "application/json"
}

data = {
    "email": "customer@example.com",
    "amount": 500000  # Amount in kobo
}

response = requests.post("https://api.paystack.co/transaction/initialize", json=data, headers=headers)
print(response.json())

5. Handling Webhooks

from flask import Flask, request

app = Flask(__name__)

@app.route('/webhook', methods=['POST'])
def webhook():
    payload = request.get_json()
    print(payload)  # Log the event
    return '', 200

6. Troubleshooting

  • 401 Unauthorized: Check your API key.
  • 400 Bad Request: Ensure required fields are provided.
  • Timeout Errors: Check your internet connection or API status.
